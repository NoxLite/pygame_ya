Суть игры: 
Вы управляете городом. Стройте новые здания, улучшайте их уровень, выполняйте различные квесты

Помощь:
-Подсказки по управлению-
Нажмите на иконку для выбора типа здания и нажмите на клавишу, соответствующую номеру здания на экране(клавиша 2 для здания с номером 2)
-Квесты-
Квесты в игре обновляются, как только будут выполнены и собраны все 4
-Сохаранение-
Сохранение происходить при каждом выходе в меню при помощи кнопки <<Выход>> внутри игрового экрана
